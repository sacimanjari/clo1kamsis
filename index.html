5. Kriptografi pada File Video (MP4) & Audio (MP3)

Agnostik Data & Looping Jutaan Blok

File MP4 sebesar 2MB murni dibaca sebagai 2.000.000 byte angka biner. Mesin JavaScript kita akan melakukan perulangan (looping) ECB/CBC sebanyak 250.000 kali putaran secara manual untuk mengacak byte piksel/audio menjadi angka tak bermakna.

Ekstensi Rusak (.cbk) & Pemulihan

Karena strukturnya sudah diacak, file tersebut akan kehilangan *header* aslinya sehingga tidak bisa diputar. Aplikasi mengemasnya menjadi file .cbk. Saat didekripsi dengan kunci yang benar, susunan byte kembali utuh, dan file bisa diputar kembali di browser secara instan.
